# Smart Event Assistant

## 🚀 Overview
An AI-powered assistant that improves physical event experiences by providing smart recommendations and guidance.

## 🎯 Features
- AI-based event suggestions
- Session guidance
- Real-time responses using Google Gemini

## ☁️ Google Cloud Integration
- Google Gemini API

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Node.js
- Google Gemini API

## ▶️ How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
Open frontend/index.html in browser

## 📌 Assumptions
- User has internet access
- API key is valid
